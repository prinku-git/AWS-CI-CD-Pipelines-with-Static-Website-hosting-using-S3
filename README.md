# AWS DEVOPS STATIC WEBSITE HOSTING WITH CI/CD PIPELINES

A fully automated CI/CD pipeline for deploying a static website using AWS services, including **GitHub**, **CodePipeline**, **CodeBuild**, and **Amazon S3** for static web hosting.

---

## FEATURES :
- GitHub-based source control  
- Automatic CI/CD pipeline using AWS CodePipeline  
- Build processing using AWS CodeBuild  
- Artifact storage in S3  
- Static website hosting on Amazon S3  
- Automatic deployment on every GitHub commit  
- IAM roles for secured service-to-service communication  

---

## SERVICES USED :
- **GitHub** (Source Code)
- **AWS CodePipeline** (CI/CD Orchestration)
- **AWS CodeBuild** (Build & Artifact Packaging)
- **Amazon S3** (Static Website Hosting + Artifact Store)
- **AWS IAM** (Service Roles & Permissions)

---

## FILES IN THIS REPO :
- Static website files (`index.html`, `styles.css`, etc.)
